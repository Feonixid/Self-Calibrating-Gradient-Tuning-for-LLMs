Self Calibrating Gradient Tuning (SGT) for Language Models

What is SGT?

When you train a language model the standard approach is to treat every word equally during training. The model gets the same gradient signal whether it already knows a word perfectly or is completely failing at it. This is wasteful. A model that has already learned common words should not be spending the same compute on them as it does on rare or difficult words it has never seen before.

SGT fixes this. It is a training controller that sits inside the optimization loop and continuously watches what the model is learning what it is forgetting, and what it is struggling with. Based on this, it reallocates gradient capacity in real time. It boosts the signal for concepts the model is losing and drops the signal for concepts it has already mastered.

The key difference from existing approaches like DoReMi is that SGT does all of this without needing a separate proxy model. It computes everything it needs directly from the main models own loss history and gradient norms in a single training pass.

Related Work and Inspirations

I want to be clear about what I built versus what already existed. SGT builds on several established ideas in machine learning. The concept of dynamic token level reweighting using a proxy model was proven by Xie et al in their 2023 DoReMi paper. The idea of focusing the optimizer on difficult or structured examples comes from Curriculum Learning by Bengio et al 2009 and Focal Loss by Lin et al 2017. Using Singular Value Decomposition SVD to extract dominant components is a classic technique in linear algebra. Token level annealing is discussed in literature as a way of dropping supervision tokens over time to encourage model exploration. Embedding noise injection draws inspiration from global regularization methods such as NEFTune by Jain et al 2023. Gradient Orthogonalization using SVD takes inspiration from Continual Learning research like Orthogonal Gradient Descent by Farajtabar et al 2020 where researchers project the gradients of new tasks to be orthogonal to the principal components of previous tasks to prevent catastrophic forgetting. Early exiting and dynamic depth derive from inference time optimization research such as DeeBERT by Xin et al 2020 where layers are skipped when the model demonstrates high confidence. SGT takes all these existing concepts and combines them in a novel way to work purely in situ without proxies.This isnt a novel idea from the ground up its pre existing concepts remixed
together in a new way to make something new and to complement that it also has novel mechanism.So in conclusion the new way of using pre exiting literature and the novel mechanism is what makes SGT novel.

Scope of Invention and Authorship

This repository introduces the following mechanisms, all of which are original work by me, Feonixid:

1 Self Calibrating Gradient Tuning SGT: A closed loop proxy free training controller that operates in situ inside the main training loop to dynamically modulate token weights and gradient scaling. Current literature like the DoReMi paper by Xie et al 2023 relies on training a separate smaller proxy model to calculate reference losses which is computationally expensive. SGT is novel because it is entirely proxy free and calculates these metrics dynamically within the main optimization loop saving massive compute while achieving the same dynamic weighting.

2 In Situ SVD Reference Geometry: SGT runs Singular Value Decomposition SVD directly on the rolling loss history and layer wise gradient norms to determine if the model is plateauing. Current papers typically use SVD for compressing weight matrices like in SVD LLM or for low rank adaptation like LoRA. This is novel because it applies SVD directly to the loss trajectories rather than the weights using spectral analysis as a real time feedback signal to control learning rate scaling which is unprecedented in optimization literature.

3 Four Quadrant Token Routing: Every token is tracked and classified into four states Drop Retain Damp and Scrub based on its loss trajectory and exposure count. Current literature explores token routing primarily within Mixture of Experts architectures like the Switch Transformer by Fedus et al 2021 or for skipping layers during inference. This is novel because it semantically classifies every single token into distinct learning states during dense model training to dynamically route gradient capacity preventing both catastrophic forgetting and wasted compute.

4 Scale Invariant Signal Extraction: SGT tracks the Spatial Coefficient of Variation and the Relative Trend converting them into Z scores using their own running statistics. Current neural network optimization papers use static hyperparameters or predefined decay schedules. The Spatial Coefficient of Variation is traditionally used in geospatial analysis and ecological data. This is novel because it brings geospatial statistics into neural network training creating a scale invariant signal extractor that dynamically adapts to any model size without requiring hardcoded threshold tuning.

5 Closed Loop Auto Annealing and Plateau Shocks: SGT smoothly transitions between aggressive and conservative training modes and triggers temporary shocks if the model gets stuck. Current training frameworks use predefined learning rate schedules like Cosine Annealing by Loshchilov et al 2016 which blindly follow a curve regardless of what the model is actually learning. This is novel because it creates an active closed loop system that detects learning plateaus in real time and automatically triggers localized optimization shocks to break out of local minima exactly when needed.

6 Targeted Embedding Shocks Token Level Annealing: This injects noise into the embedding layer exclusively for dynamically identified irreducible tokens during training. Current literature like the NEFTune paper by Jain et al 2023 explores embedding noise injection but applies it globally to all tokens as a blanket regularization technique. This is novel because it utilizes token specific noise injection as an active feedback mechanism to selectively shock only the stuck tokens out of local minima without disrupting the representations of already learned concepts.

7 Eigen Gradient Orthogonalization: This applies orthogonalization dynamically mid training on a per token basis derived from the loss trajectory SVD. Current papers on Continual Learning like Orthogonal Gradient Descent by Farajtabar et al 2020 project the gradients of new tasks to be orthogonal to previous tasks to prevent catastrophic forgetting. This is novel because it repurposes continual learning math to mathematically lock in solved tokens in the embedding space during standard pre training transforming gradient orthogonalization into a token level auto scaling optimizer.

8 SGT Driven Early Exiting: This introduces token level early exiting as a training time optimization bypassing deep Feed Forward layers while retaining tokens as Keys and Values. Current literature like the DeeBERT paper by Xin et al 2020 and the ADEPT framework strictly use early exiting and dynamic depth as inference time optimizations to make text generation faster. This is novel because it routes compute dynamically during training based on SGT feedback whereas existing research focuses solely on speeding up post training inference.

9 Attention Entropy Guided Auto Scaling: This monitors the Shannon entropy of attention heads during the forward pass and uses it as a feedback signal to dynamically scale the learning rate of each layer independently. Current research like the ALAER paper by Wang et al 2025 uses attention entropy as a static penalty term added to the loss function to prevent entropy collapse. This is novel because it transforms attention entropy from a static regularization penalty into a real time closed loop feedback signal for a dynamic layer wise learning rate auto scaler which actively forces plasticity when heads become too rigid.

10 Token Quadrant Gradient Conflict Surgery: This dynamically resolves gradient conflicts between struggling tokens and solved tokens within a single training batch by projecting out conflicting mathematical components. Current literature such as the PCGrad paper by Yu et al 2020 projects conflicting gradients to be orthogonal but strictly does this between different tasks in multi task learning settings. This is novel because it performs gradient surgery internally based on token quadrant states projecting gradients within the exact same task during standard pre training to ensure reinforcing solved tokens never damages the optimization of struggling tokens.

11 SVD Driven Adaptive Batch Composition: This controls the training data loader to actively construct batches that maximize exposure to tokens in the retain and damp quadrants while minimizing solved tokens. Current frameworks like Curriculum Learning by Bengio et al 2009 statically sort datasets by difficulty before training begins. This is novel because it creates a closed loop active data selection mechanism that dynamically alters the training distribution on the fly using real time feedback from the SVD loss trajectory ensuring computational resources are exclusively spent on sequences dense with struggling concepts.
